# 🏹 P2PPong v4.0

**Транспортный арбитр для RobinHood P2P. Бета-версия.**

P2PPong обеспечивает установку защищённых каналов связи между пользователями RobinHood P2P. Поддерживает два сервера на выбор: Render (везде) и Cloudflare Workers (с VPN).

Не хранит сообщения, не участвует в переписке. Только создаёт и верифицирует каналы связи.

---

## Как работает

| Компонент | Назначение |
|-----------|------------|
| **Render** | Основной сервер маяков (работает везде) |
| **Workers** | Быстрый сервер (требует VPN/Amnezia) |
| **AES-256-GCM** | Шифрование сигнального канала |
| **SHA-256** | Генерация сессионных ключей |

---

## Стрелы

| Стрела | Сервер | Скорость | Где работает |
|--------|--------|----------|--------------|
| 🏕️ **Шервуда** | Render | 2-6 сек | Везде |
| ⚡ **Варп** | Workers | <1 сек | С VPN/Amnezia |

## Условия использования

P2PPong является служебным компонентом RobinHood P2P. На него распространяются все условия использования RobinHood P2P. Автор не несёт ответственности за использование P2PPong третьими лицами в любых целях.
